Detailed Description

APPLICATION: Gives structures and main functions to create random belief functions.

This module provides with methods to randomly generate mass functions. It is roughly based on common algorithms to do so. An interesting paper on the subject is "Random Generation of Mass Functions: A Short Howto" by Thomas Burger and SÃ©bastien Destercke (2012).
